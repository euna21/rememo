# re:memo 📝

함께한 하루를 여러 사람이 각자의 감각으로 기록하고, AI가 이를 하나의 결과물로 완성해 주는 협업형 다이어리 앱 프로젝트입니다

## 🖥️ 서비스 소개
- **함께 만드는 공동 기억 아카이브**: 시간이 지나도 당시의 감정과 분위기를 생생하게 떠올릴 수 있는 특별한 기억 저장소를 제공합니다.

- **역할 기반의 새로운 기록 방식**: 타임 캡슐을 통해 부여된 각자의 역할에 맞춰 하루를 기록합니다.

- **AI가 완성하는 협업형 다이어리**: 친구들과 함께 수집한 기록을 Claude API가 통합 분석하여 하나의 다이어리로 자동 생성하며, 완성된 결과물은 함께 꾸미고 외부로 공유할 수 있습니다.

## 📁 폴더 구조 (Project Structure)

코드를 이해하기 쉽도록 기능별로 폴더를 나누어 관리하고 있습니다.

* `src/screens/` : 각 화면의 전체 UI 구성 요소 (로그인, 책장, 다이어리 등)
* `src/components/` : 버튼, 사이드바 등 여러 화면에서 재사용하는 공통 부품
* `src/data/` : 앱에서 사용하는 데이터 모음 (mockData.ts)
* `src/types/` : 데이터 타입 및 인터페이스 설계도 (TypeScript 정의)
