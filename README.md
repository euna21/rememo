# re:memo 📝

추억을 기록하는 모바일 다이어리 앱 프로젝트입니다.

## 📁 폴더 구조 (Project Structure)

코드를 이해하기 쉽도록 기능별로 폴더를 나누어 관리하고 있습니다.

* `src/screens/` : 각 화면의 전체 UI 구성 요소 (로그인, 책장, 다이어리 등)
* `src/components/` : 버튼, 사이드바 등 여러 화면에서 재사용하는 공통 부품
* `src/data/` : 앱에서 사용하는 데이터 모음 (mockData.ts)
* `src/types/` : 데이터 타입 및 인터페이스 설계도 (TypeScript 정의)
